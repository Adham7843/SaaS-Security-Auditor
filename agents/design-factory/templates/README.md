## Usage

### Fetching More Prompts

```bash
# Get full prompt for specific style
superdesign get-prompts --slugs "news-print,glassmorphism-style" --json

# Batch fetch (max 20 per request)
superdesign get-prompts --slugs "style1,style2,..." --json
```

### Using in Design Factory

These prompts can be used as:

1. **Design Direction** — Reference entire design systems
2. **Component Inspiration** — Mix tokens from different styles
3. **Landing Page Templates** — Full page layouts
4. **Background Animations** — WebGL/Canvas effects
5. **Interactive Elements** — Buttons, loaders, modals
