## Dialogue Patterns

### Phrases That Signal Approval

| Phrase | Interpretation |
|--------|----------------|
| "I like it" | Add to positive examples |
| "This works" | Record as approved pattern |
| "Yes, exactly" | Strong approval |
| "Keep this" | Lock this decision |
| "This feels right" | Tone matched |

### Phrases That Signal Rejection

| Phrase | Interpretation |
|--------|----------------|
| "This feels wrong" | Add to negative examples |
| "I don't like this" | Reject pattern |
| "Too [X]" | Adjust in direction |
| "Not my vibe" | Tone mismatch |
| "It's okay" | Neutral - don't push |

### Directional Feedback

| Phrase | Adjustment |
|--------|------------|
| "Too playful" | More serious, professional |
| "Too serious" | Add warmth, playfulness |
| "Too busy" | Reduce detail, more white space |
| "Too plain" | Add visual interest |
| "Make it pop" | Increase contrast, add accent |
| "Tone it down" | Reduce contrast, simpler |
| "More premium" | Add luxury cues (gold, black, serif) |
| "More friendly" | Rounder, warmer, casual |
